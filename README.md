# FitForge 💪

Personalisierte Trainingsapp für Sportler aller Levels.

## Beschreibung
FitForge erstellt individuelle Trainingspläne basierend auf Körperdaten,
Fitnesszielen und verfügbarer Ausrüstung. Mit über 105 Übungen in 7 Kategorien
– von Körpergewicht bis Boxen und Thaiboxen.

## Features
- 🤖 KI-generierte Trainingspläne (Anthropic API)
- ⚡ Regelbasierte Planerstellung (105+ Übungen)
- ⏱️ Boxtimer mit Runden und Pausen
- 📱 PWA – als App installierbar
- 🔐 Registrierung und Login mit Spring Security

## Technologien
- Java 26 + Spring Boot 3.5
- Spring Security + Spring Data JPA
- MySQL + Hibernate
- Thymeleaf + HTML/CSS
- Gradle

## Installation
1. MySQL starten (XAMPP)
2. `application.properties` aus `application.properties.example` erstellen
3. `.\gradlew.bat bootRun` ausführen
4. Browser öffnen: `http://localhost:9090`
